Project 4 - Generalized Additive Model (GAM) and Nadaraya-Watson¶

Sometimes, data is not linear, so using linear models will not provide the most accurate models. Generalized Additive Models (GAMs) and Non-Parametric models are two ways to model non-linear data, while still maintaining explainability.

Linear Models

Linear models are modeled with the equation: $\hat{y} = \beta_0x_0 + \beta_1x_1 + ... + \beta_nx_n$, where $\hat{y}$ is the conditional expectation of $x$ and $y$, and the $\beta$s are weights that explain the relationship between each input feature and the output.

Linear Generalized Additive Model

Where linear models have a beta to demonstrate the relationship between the input features and the outputs, linear generalized additive models use functions: $\hat{y} = \beta_0 + f_1(x_1) + f_2(x_2) + ... + f_n(x_n)$, where each of the functions have their own set of trainable weights.

Nadaraya-Watson

The Nadaraya-Watson model is a non-parametric model, similar to locally weighted regression. It predicts outputs by assigning weights to nearby inputs based on the kernel and bandwidth parameters.

Compare Models with KFold¶

In [ ]:
kf = KFold(n_splits=10, shuffle=True, random_state=42069)

gam_mse = []
nw_mse = []

scale = QuantileTransformer(n_quantiles=8)

gam = LinearGAM(n_splines=37)
nw = NadarayaWatson(degree=1, gamma=29.150530628251758, kernel='rbf')

for idxtrain, idxtest in kf.split(x):
  xtrain_kf = x[idxtrain]
  ytrain_kf = y[idxtrain]
  ytest_kf = y[idxtest]
  xtest_kf = x[idxtest]
  xtrain_kf = scale.fit_transform(xtrain_kf)
  xtest_kf = scale.transform(xtest_kf)

  gam.fit(xtrain_kf,ytrain_kf)
  gam_yhat = gam.predict(xtest_kf)

  nw.fit(xtrain_kf, ytrain_kf)
  nw_yhat = nw.predict(xtest_kf)

  gam_mse.append(mse(ytest_kf, gam_yhat))
  nw_mse.append(mse(ytest_kf, nw_yhat))

print('The Cross-validated Mean Squared Error for Gradient Boosting is : ' + str(np.mean(gam_mse)))
print('The Cross-validated Mean Squared Error for Random Forest is : ' + str(np.mean(nw_mse)))
